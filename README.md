# 🏭 Manufacturing Execution System (MES) — Machine Tracking Platform

> **Status: ✅ Live in Production** — Running 24/7 on LAN across 5 factory departments

## About
A full-stack Industrial MES application built and deployed solely by one developer.
Digitises the complete product lifecycle — from order entry to dispatch —
and tracks machines in real time across a live manufacturing environment.

## Key Features
- 🔧 Real-time machine status tracking (uptime, downtime, cycle count)
- 📦 Product lifecycle management: Order → Planning → Production → QC → Dispatch
- 📊 Live KPI dashboards tracking 10+ metrics (OEE, utilisation, defect rate)
- 📷 QR code scanning automation at production checkpoints
- 🔐 JWT authentication + Role-Based Access (Admin, Production, QC, Dispatch)
- 📈 Automated monthly performance reports
- ☁️ Deployed on AWS (EC2, S3, RDS, CloudWatch) + LAN
